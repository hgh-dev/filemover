# File Mover (파일 무버)

**File Mover**는 언제 어디서나 간편하게 파일, 사진, 메모, 링크를 업로드하여 임시 보관하고 어느 기기에서나 다운로드할 수 있는 웹 애플리케이션입니다. 사무 업무 시 간단한 사진, 파일, 메모, 링크를 다른 기기로 간편하게 옮기기 위해 제작되었습니다. 모바일과 데스크톱 환경 모두에서 쾌적하게 사용할 수 있습니다.

---

## 주요 기능

### 1. 다양한 데이터 관리
*   **파일 업로드**: 다양한 형식의 일반 파일을 빠르고 안전하게 서버에 업로드합니다.
*   **사진 업로드 및 최적화**: 다중 이미지 업로드를 지원하며, 업로드 전 해상도 조절(리사이징) 기능을 제공하여 용량을 절약할 수 있습니다.
*   **메모 작성**: 짧은 텍스트나 긴 아이디어를 빠르게 기록하고 백업합니다. 
*   **링크 저장**: 웹페이지 주소를 저장하고, 클릭 한 번으로 빠르게 이동할 수 있습니다.

### 2. 구글 계정 연동 (Firebase Auth)
*   복잡한 회원가입 절차 없이 **Google 계정 로그인을 지원**하여 원클릭으로 안전하게 접속합니다.
*   사용자별로 데이터를 완벽하게 격리하여 안전한 프라이빗 저장소를 제공합니다.

### 3. 실시간 동기화 상태 및 클린 시스템
*   **기간 만료 자동 정리 (용량별 보관 기한)**: 파일 크기에 따라 자동으로 만료 기한이 차등 설정되며, 기간이 지난 파일은 서버에서 깔끔하게 자동 파기됩니다. 만료 48시간 전부터는 빨간색 경고 문구가 나타나 기한을 연장할 수 있습니다.
    *   `50MB 이상` 파일: **3일 보관**
    *   `10MB 이상 50MB 미만` 파일: **7일 보관**
    *   `10MB 미만` 일반 파일/사진: **30일 보관**
    *   `메모 및 링크`: **영구 보관** (기간 만료 없음)
*   **휴지통 및 비정상 데이터 정리**: 업로드 또는 삭제 과정에서 네트워크 중단이나 비정상적인 종료가 발생해도 안심하세요. 다시 접속 시 알아서 찌꺼기 파일을 클라우드에서 완벽하게 제거합니다.
*   **고스트 카드 방지**: 서버에 실체 파일이 없는 데이터(고스트 데이터)를 자동 감지하고 삭제하는 자가 치유 기능을 탑재하고 있습니다.


### 4. 세련된 파일 목록 및 필터링 기능
*   방식이 다른 데이터들을 한눈에 알아볼 수 있도록 직관적인 UI(배지, 아이콘)를 제공합니다.
*   상단의 칩버튼을 눌러 전체, 메모, 링크, 사진, 파일만 각각 따로 **필터링**하여 직관적으로 확인이 가능합니다.
*   사진 및 파일 다건 업로드 시 하나의 카드로 깔끔하게 그룹핑(외 N건) 되며, 클릭 시 팝업을 통해 한 번에 묶어서 다운로드를 관리할 수 있습니다.

### 5. PWA (Progressive Web App) 완벽 지원
*   별도로 앱 스토어에서 다운로드하지 않고도 모바일이나 PC 화면에 홈 화면 아이콘을 추가하여 **네이티브 앱처럼 사용**할 수 있습니다.
*   자체 서비스 워커 구조로 리소스 캐싱을 통해 더 빠르고 쾌적한 로딩 속도를 경험할 수 있습니다.

---

## 🛠️ 기술 스택 (Tech Stack)

*   **프론트엔드**: HTML, CSS(Vanilla), JavaScript(ES6 모듈식 구조화)
*   **백엔드/인증**: Firebase Auth, Firebase Firestore (NoSQL 실시간 데이터베이스)
*   **미디어 서버**: Cloudinary (빠른 이미지/동영상/파일 업로드 및 관리, 자동 포맷 변환)
*   **기타**: PWA Service Worker 기반 리소스 캐싱 적용

## 💡 요약
**File Mover**는 언제든지 자료를 올려두었다가 필요할 때 꺼내쓰는 "개인용 클립보드 및 클라우드" 역할을 충실히 수행하도록 설계되었습니다. 깔끔한 정리 기능, 불필요한 서버 자원을 방치하지 않는 꼼꼼한 청소 알고리즘이 포함된 훌륭한 시스템 스크립트를 보유하고 있습니다.
